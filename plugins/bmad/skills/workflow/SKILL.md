# Workflow Workflow

**Goal:** Create, edit, validate, or rework BMAD workflows.

**Agent:** Workflow Builder (Wendy) **Module:** BMB

---

## Execution

Select a sub-workflow based on user intent:

| Trigger | Sub-workflow                                    | Description                        |
| ------- | ----------------------------------------------- | ---------------------------------- |
| CW      | `./workflow-create-workflow.md`                  | Create a new BMAD workflow         |
| EW      | `./workflow-edit-workflow.md`                    | Edit an existing workflow          |
| VW      | `./workflow-validate-workflow.md`                | Validate against best practices    |
| MV      | `./workflow-validate-max-parallel-workflow.md`   | Validate in MAX-PARALLEL mode      |
| RW      | `./workflow-rework-workflow.md`                  | Rework to V6-compliant version     |

Read and follow the selected sub-workflow file.
